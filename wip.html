About the roadmap columns

  • Backlog: recorded long-term ideas, in no particular order
  • Soon™: items intended for short-term to medium-term
  • In Progress: items currently being worked on
  • Available in beta: items available in the beta versions
  • Generally available: items available in the production versions

How to participate

To participate, you will need to create yourself a Github account if you don't already have one. You can suggest new ideas or vote on existing ideas you would like to priorizes. To participate, first check if your idea is already listed here and then:

  • If an item with your idea already exists, add a "👍" reaction on it to vote for it
  • Otherwise, you can add the idea as a new issue here

Note that items with more votes are more likely to be prioritized.
